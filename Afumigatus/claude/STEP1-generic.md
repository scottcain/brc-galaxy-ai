# Connect to Galaxy

## Prerequisites

To begin the analysis, we need to connect to a Galaxy instance for which we need three prerequisites:

- a Galaxy API key stored in a keychain so this AI agent has access to it: stored in the root directory in a file called galaxy-key.txt 
- access to a set of skills and MCP guidelines for describing how AI agent will interact with Galaxy: accessible here https://github.com/galaxyproject/galaxy-skills
- a Galaxy history in which we will perform all analyses: https://usegalaxy.org/u/scottcain/h/a-fumigatus-antifungal-resistance-analysis 

## Execution

- Do not write any notes or comments in this file; place execution notes in a file called STEP1-execution-notes.md.
- Read the Prerequisites section above
- Check that Galaxy API key works
- Consult https://github.com/galaxyproject/galaxy-skills to read the skills and understand how to install MCP Server.
- Connect to Galaxy and print how many data sets you see in the history linked in the prerequisites section.
